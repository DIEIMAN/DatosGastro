# Oportunidades detectadas

## 1. Padrón gastronómico vivo
Problema: no hay padrón vivo consolidado. Datos: AGC + turismo + inspecciones + USIG. MVP: tabla actualizable por CUIT/dirección/rubro.

## 2. Monitor de altas gastronómicas
Problema: no se ve dinámica reciente del sector. Datos: Habilitaciones AGC 2015-2026. MVP: dashboard mensual por barrio/comuna/rubro.

## 3. Agenda gastronómica estructurada
Problema: eventos en HTML/noticias. Datos: agenda GCBA + cámaras. MVP: `fact_evento_gastronomico` con API/CSV público.

## 4. Sistema gratuito para restaurantes
Problema: comercios chicos no tienen herramientas simples de gestión. MVP: app web/mobile para mesas, estados, reservas y turnos.
